DarkSys 🕶️

DarkSys is an advanced hacker-style Termux tool designed for fun and system monitoring. It features a matrix animation, fake hacking simulation, and useful system information utilities with a stylish command-line interface.

---

🔥 Features

- 🟢 Matrix Effect (Hacker Style Animation)
- 💻 Fake Hacking Simulation Screen
- 📊 System Information (Device & OS)
- ⚡ CPU Usage Info
- 🧠 RAM Usage Info
- 💾 Storage / Disk Info
- 🌐 Network Information
- 🎨 Stylish CLI Interface with colors
- 👨‍💻 Developed by Jugnu

---

📦 Installation (Termux)

Run the following commands in Termux:

pkg update && pkg upgrade
pkg install python git
pip install rich pyfiglet

---

🚀 How to Run

git clone https://github.com/yourusername/darksys.git
cd darksys
python darksys.py

---

⚡ Run as Command (Optional)

You can run this tool directly as a command:

chmod +x darksys.py
mv darksys.py $PREFIX/bin/darksys

Then simply run:

darksys

---

🧪 Preview

When you run the tool, you will see:

- Matrix style green falling code animation
- Fake hacking terminal simulation
- Stylish DarkSys banner
- Menu-driven system tools
